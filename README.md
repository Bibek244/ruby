## Installation

To clone and run this project, you'll need have Git installed on your computer. From your command line:

# Clone this repository
```bash
git clone https://github.com/Bibek244/ruby.git
```
# Go into the repository
```bash
cd ruby
```
# Run program
```bash
#example
cd MasterMind
ruby Mastermind.rb
```
